Register

To add a course to this list, find the course on the Available Courses page and select 'Register'

To remove a course from the list select the Remove button next to the course you wish to remove.

The courses listed will be requested for registration by selecting the 'Register' button.

After selecting Register you will see either a checkmark or an X next to each course showing if the class was registered for successfully.
